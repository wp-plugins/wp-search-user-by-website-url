A plugin which provide search by user's website's url in admin users listing.

== Description ==
This plugin adds a additional column in admin user's listing with user's website url and allows search by website url.


== Installation ==
= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Search User By Website Url'
3. Activate 'WP Search User By Website Url' from your Plugins page. 

= From WordPress.org =

1. Download 'WP Search User By Website Url'.
2. Upload the 'wp-search-user-website' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'WP Search User By Website Url' from your Plugins page.
